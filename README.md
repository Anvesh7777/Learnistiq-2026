# Learnistiq 

A modern full-stack Learning Management System (LMS) built using the MERN stack with secure authentication, course management, payments, certificates, progress tracking, and admin analytics.

## 🌐 Features

### Student Features

* User Authentication (Signup/Login)
* Forgot Password & Reset Password
* Browse Courses
* Course Details Page
* Course Reviews & Ratings
* Add to Cart
* Razorpay Payment Integration
* Purchase History
* Course Progress Tracking
* Watch Lectures
* Course Completion Certificates
* Public Certificate Verification
* User Profile Management

### Admin Features

* Admin Authentication
* Create Courses
* Update Courses
* Delete Courses
* Manage Lectures
* Upload Videos & PDFs
* Analytics Dashboard
* Revenue Tracking
* Enrollment Tracking
* Certificate Statistics
